# TyG and HOMA-IR Calculator

Educational web-based calculator for the triglyceride–glucose (TyG) index and HOMA-IR.

This tool provides evidence-based orientation regarding metabolic syndrome risk using surrogate metabolic indices.  
It does NOT provide diagnosis or medical advice and does not replace clinical evaluation by a physician.

## Features
- TyG index calculation
- HOMA-IR calculation
- Support for mg/dL and mmol/L units
- Evidence-based interpretation
- Educational disclaimers

## Disclaimer
This application is intended for educational and research purposes only.  
Always consult a qualified healthcare professional for medical decisions.
